# 契约：位置语义与退出完成判定

## 目标

定义位置相关条件、段完成判定和退出完成判定的统一业务语义。

## 位置语义契约

| 语义 | 来源 | 说明 |
|------|------|------|
| 到车头 | 平台位置事实 + 领域判定 | 表示工步目标到达车头边界 |
| 到车尾 | 平台位置事实 + 领域判定 | 表示工步目标到达车尾边界 |
| 相对距离达到 | 平台位置事实 + 领域判定 | 表示相对车头/车尾的距离阈值满足 |
| 绝对位置达到 | 平台位置事实 + 领域判定 | 表示龙门绝对位置满足阈值 |
| 回零完成 | 平台反馈 + 领域判定 | 表示回零退出动作完成 |

## 判定边界契约

| 判定类型 | 含义 | 是否代表整段结束 |
|------|------|------|
| 本段完成 | 本段运行目标已满足 | 否 |
| 收尾完成 | 退出动作全部完成 | 是 |
| 段内超时 | 运行目标未按时满足 | 否，进入异常路径 |
| 收尾超时 | 退出动作未按时完成 | 否，进入退出失败路径 |

## 条件控制契约

1. 每个条件控制必须声明使用相对位置还是绝对位置。
2. 开始喷药和停止喷药必须来自同一种位置语义解释。
3. 退出阶段开始后，不允许未声明 `active_during_exit` 的条件控制继续悬空生效。

## 退出动作契约

| 动作 | 完成要求 |
|------|------|
| 停刷 | 收到停刷完成反馈或达到明确定义的完成条件 |
| 停药 | 药剂通道确认关闭 |
| 顶刷回零 | 回零完成反馈成立 |
| RO 水关闭 | 冲洗输出确认关闭 |
| 风干关闭 | 风机输出确认关闭 |

## 验证准则

- 能解释为什么在某个位置开始喷药、在某个位置停止喷药。
- 能区分“已到目标位置但仍在收尾”和“整段真正结束”。
- 位置信号丢失、回零失败、关闭失败都会进入固定异常路径。
